Snoubolr AWS-ready unique static site

Что внутри:
- уникальный дизайн, цвета, SVG-графика и иконки
- Home, Product, Demo, Pricing, Roadmap, AWS Vision, About, Contact
- Security, Privacy, Terms, Blog/Updates, Changelog, Waitlist
- CNAME для GitHub Pages: snoubolr.online
- robots.txt и sitemap.xml

Как залить:
1. Распаковать архив.
2. Заменить файлы в GitHub репозитории snoubolr.
3. Commit changes в main branch.
4. Проверить https://snoubolr.online

Важно: сайт делает бизнес-информацию более полной, но не гарантирует одобрение AWS Activate. Для повторной заявки используйте только реальные и консистентные данные.
