SNOU BOLR — AI ANALYTICS STARTUP LANDING

Эта версия сайта специально переделана так, чтобы было понятно:
- Snoubolr использует AI для аналитики стартап-проектов.
- AI анализирует идею, аудиторию, рынок, стадию, tech stack и цели проекта.
- AI помогает развивать проекты: предлагает roadmap, MVP next steps, похожие стартапы и полезных людей.
- Есть отдельная страница AI Analytics: analytics.html.
- Есть AWS Vision для объяснения cloud/AI/analytics направления.

Что внутри:
- index.html — главная страница
- analytics.html — страница про AI аналитику
- features.html — функции продукта
- community.html — комьюнити
- roadmap.html — план развития
- aws-vision.html — cloud/AWS направление
- contact.html, waitlist.html, login.html, blog.html
- assets/css/styles.css — дизайн и анимации
- assets/js/script.js — меню, анимации, видео, формы через email
- assets/img — картинки и SVG диаграммы
- assets/video/snoubolr-demo.mp4 — активное видео
- CNAME — домен snoubolr.online

Как загрузить на GitHub Pages:
1. Распакуй ZIP.
2. Открой папку snoubolr_ai_analytics_landing.
3. Загружай на GitHub все файлы и папки ИЗНУТРИ этой папки, не сам ZIP.
4. Файл index.html должен лежать в корне репозитория.
5. В GitHub: Settings → Pages → Deploy from branch → main → root.
6. В Custom domain укажи snoubolr.online.

Важно:
Не удаляй DNS записи Zoho в GoDaddy, иначе сломается почта admin@snoubolr.online.
